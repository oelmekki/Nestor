# Nestor
A simple Ruby FTP server

### Install
    - install rvm
    - rvm get head
    - rvm use 2.2.2
    - gem install bundler
    - git clone this repository
    - cd into the project's directory
    - bundle install (to install depedencies)

### Use
Just type

    ruby volcano_ftp.rb <port>  

That's it, enjoy your FTP server !
